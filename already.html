First up, open a new window.

You'll want to see these instructions clicking back and forth.

Go to the website you bought your domain name from. Sign in.

Find the section listing the domain(s) you have already.

It changes from site to site, but it'll be something along the lines of "my domains" / "manage my domains".

Pick the one you want and select edit/manage/update/change DNS records.

There should be four fields to fill in:

  • "Host"
  • "Type"
  • "Value" (or "Points to")
  • "TTL"

Set "Type" to "A" and "Value" to "192.30.252.153".

Leave everything else on the page blank and save these changes.

Do the same again, but this time add "192.30.252.154" as your "Value". Save these changes.

Log out and close the window. That's the hard bit over.
